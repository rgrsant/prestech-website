Firewalls são essenciais para a segurança de seus computadores

Firewalls protege o seu sistema de situações indesejadas, ele faz isso por meio de análises profundas de tráfego de rede, utilizando um conjunto de regras para determinar se os dados que são recebidos e enviados são legítimos. Os pacotes de arquivos que não corresponder a essas regras, são automaticamente bloqueados.


O conceito de um firewall pode ser comparada à portaria de condomínio, onde é preciso obedecer as regras impostas e estar apto para entrar naquele local. O firewall é capaz de proibir uma série de ações maliciosas, como um malware: um programa que envia dados sigilosos para a Internet na tentativa de obter acesso à rede a partir de computadores externos não autorizados.
